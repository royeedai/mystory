# FactoryBeanObjectType 错误修复说明

## 问题描述
应用启动时出现错误：
```
Invalid value type for attribute 'factoryBeanObjectType': java.lang.String
```

## 根本原因
MyBatis-Spring 2.1.2版本不兼容Spring Boot 3.x（Spring Framework 6.1）。
- MyBatis-Spring 2.x 支持 Spring Framework 5.x（Spring Boot 2.x）
- MyBatis-Spring 3.x 支持 Spring Framework 6.x（Spring Boot 3.x）

## 修复方案
在 `novel-api/pom.xml` 中进行了以下修改：

1. **排除旧版MyBatis-Spring依赖**
   - 从mybatis-plus-boot-starter中排除了mybatis-spring 2.1.2

2. **显式添加兼容版本**
   - 添加了mybatis-spring 3.0.3依赖

## 修改的文件
- `/workspace/novel-api/pom.xml`

## 已更新的JAR包
- `/workspace/public/jar/novel-api-1.0.0.jar`
- `/workspace/public/jar/novel-api.jar`

## 验证结果
✅ 应用启动成功
✅ Spring Boot 3.2.0 正常运行
✅ MyBatis Plus 3.5.5 正常工作
✅ Tomcat服务器在端口8080启动，上下文路径为/api
